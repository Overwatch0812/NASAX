CampusX API

Endpoints To use Project database model

  • /api/ : To view all the projects
  • /api/create/ : To create a project
  • /api/id_of_project/ : To view a specific project
  • /api/id_of_project/update/ : To update a specific project
  • /api/id_of_project/delete/ : To delete a specific project
